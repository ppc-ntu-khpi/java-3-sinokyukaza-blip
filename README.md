# Перевірка числа Армстронга

## Опис
Програма дозволяє користувачу ввести число з клавіатури
та перевіряє, чи є воно числом Армстронга.

## Алгоритм
1. Підрахувати кількість цифр.
2. Піднести кожну цифру до степеня кількості цифр.
3. Знайти суму.
4. Порівняти результат з початковим числом.

## Приклад

Введення:
168

Вивід:
Число 153 не є числом Армстронга.
